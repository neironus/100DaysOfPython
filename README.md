### Day 48
    Mesuring perf with cProfile
    ```python
    import cProfile
    profiler = cProfile.Profile()
    profiler.disable()

    def main():
        profiler.enable()
        profiler.disable()

    if __name__ == '__main__':
        profiler.print_stats(sort='cumtime')
    ```
